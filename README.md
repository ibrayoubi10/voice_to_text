# 🎙️ Local Voice-to-Text Recorder

A lightweight local desktop application that records audio from the microphone and converts speech to text using Whisper.
The application runs entirely locally without external APIs.

It allows users to:

- record voice using a simple interface
- automatically convert speech to text
- save the transcription as a .txt file

# 📌 Future Improvements
Planned improvements include:
- real-time transcription
- automatic daily summaries
- task extraction from speech
- note organization by date
- exporting structured notes
